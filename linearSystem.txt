Ax = b 라는 정형적인 방법으로 표현하고 해결하는 방법

용어 정리
Equations(E + 아래첨자)로 식을 표현
unknowns or variable(미지수 or 변수)
3 x 3 = 식의 개수 x 미지수의 개수

미지수의 승수가 1로만 구성되어 있으면 선형방정식
sin함수가 붙어 있거나 승수가 1이 아니면 비선형방정식(식에 xy 이런식으로 있을때 xy 지수가 1이라도 연산할때 지수끼리 더해지기 때문에 비선형방정식이 된다)

선형시스템을 Ax = b 로 표현하기 (볼드체로 표현)

3x + y = 2                       [ 3   1 ]   [ x         [2]
x - 2y = 3            ->         [ 1  -2 ]            =  [3] 
2x - 4y = 6                      [ 2  -4 ]     y ]       [6]

m x n 선형시스템의 Ax = b 표현을 정리

식은 행이고, 행은 식이다.
m은 linear equation의 개수이다.
n은 unknown의 개수이다
A 는 m x n 행렬이다.
x 는 n- 벡터이다.
b 는 m- 벡터이다.
